# Local AI Lab Public Boundary

Status: scaffolded
Publication state: public scaffold
Release state: not released

## Purpose

This boundary defines what can be said publicly about a local AI lab without exposing private infrastructure or active operations.

## Public-Safe Material

Allowed material:

- synthetic lab-publication decision rules;
- artifact classification examples;
- public/private/sealed labels;
- no-topology documentation patterns;
- human review gates for automation, model, and agent claims.

## Held Out

Do not include real devices, topology, hostnames, mesh or VPN details, endpoints, ports, private URLs, credentials, tokens, private source paths, private agents, active agent internals, tool-permission maps, private logs, live automation, production workflows, deployment claims, or production-readiness claims.

## Public Note Standard

A public lab note must be useful as a boundary example even when every real device, network, credential, and workflow detail is absent.
